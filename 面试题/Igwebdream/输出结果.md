```js
// 写出打印结果，并解释为什么
var a = { k1: 1 };
var b = a;
a.k3 = a = { k2: 2 };
console.log(a); // {k2: 2}
console.log(b); // {k1:1 k3: {k2: 2}}
```

此题出现a.k3 = a = { k2: 2 }; **点运算**的运算等级高于 = ，所以a.k3的运算高于a={k2:2}
